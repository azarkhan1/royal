# حسابدار من - Personal Accounting Platform

یک پلتفرم مدیریت مالی شخصی با پشتیبانی کامل از راست‌چین (RTL) و زبان فارسی.

## ویژگی‌ها

- ✅ **رابط کاربری RTL**: طراحی شده برای فارسی‌زبانان
- ✅ **حالت تاریک**: پشتیبانی کامل از تم تاریک
- ✅ **چند ارزی**: پشتیبانی از AFN, IRR, USD, EUR
- ✅ **داشبورد هوشمند**: نمودارها و آمار لحظه‌ای
- ✅ **مدیریت فاکتور**: ایجاد، ویرایش و ارسال فاکتور
- ✅ **مدیریت هزینه‌ها**: دسته‌بندی و ردیابی هزینه‌ها
- ✅ **گزارش‌های مالی**: گزارش‌های جامع و بصری
- ✅ **مدیریت مشتریان**: اطلاعات و تاریخچه مشتریان
- ✅ **محصولات و خدمات**: مدیریت کاتالوگ
- ✅ **پشتیبان‌گیری**: دانلود و بازیابی داده‌ها

## تکنولوژی‌ها

- **React 18** با TypeScript
- **Vite** برای ساخت سریع
- **Tailwind CSS** برای استایل‌دهی
- **React Router** برای مسیریابی
- **Recharts** برای نمودارها
- **Lucide React** برای آیکون‌ها

## نصب و راه‌اندازی

```bash
# نصب وابستگی‌ها
npm install

# اجرای محیط توسعه
npm run dev

# ساخت برای تولید
npm run build

# پیش‌نمایش نسخه تولید
npm run preview
```

## ساختار پروژه

```
src/
├── components/       # کامپوننت‌های قابل استفاده مجدد
├── contexts/        # Context API برای مدیریت state
├── layouts/         # لایه‌بندی صفحات
├── pages/           # صفحات اصلی برنامه
└── index.css        # استایل‌های全局
```

## صفحات

### صفحات عمومی
- **Landing Page**: صفحه اصلی با معرفی ویژگی‌ها
- **Login**: صفحه ورود
- **Signup**: صفحه ثبت‌نام
- **Onboarding**: راهنمای شروع (۴ مرحله)

### صفحات برنامه
- **Dashboard**: داشبورد اصلی با آمار و نمودارها
- **Invoices**: مدیریت فاکتورها
- **Expenses**: مدیریت هزینه‌ها
- **Products**: محصولات و خدمات
- **Clients**: مدیریت مشتریان
- **Reports**: گزارش‌های مالی
- **Settings**: تنظیمات
